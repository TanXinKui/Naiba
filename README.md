# 项目介绍：
奶爸，原来是大三的时候做的一个综合项目，目的是餐厅店里用的点餐app。这个是客户端，用于客人点餐用的。
之前做的时候没有前后端分离，是直接用jdbc连接数据库，现在把前端和后端分离了出来。
后端部署在本人的腾讯云服务器（在eclipse打包成war后部署在tomcat）。
